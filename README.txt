Задача: 
Написать программу, которая из имеющегося массива строк формирует массив строк, длина которых меньше либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма.

Алгоритм решения:
Проходим весь массив со строками и смотрим на длину строки, если она меньше либо равна 3,
то записывает эту строчку в новый массив, и увеличиваем переменую, которая отвечает за количество слов во втором массиве, увеличиваем на 1.
Выводим второй массив на экран.
